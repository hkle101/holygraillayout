Harmful Software

Learn about different kinds of harmful software — viruses, trojans, ransomware, and spyware — how they behave, and basic steps to prevent and recover from infections. Click each box title to read details and mitigation tips.

Viruses & Worms

Viruses attach to files and require execution to spread; worms self-propagate across networks. Both can corrupt data and consume resources. Keep systems patched and limit unnecessary network exposure to reduce risk.

Trojans & Backdoors

Trojans disguise malicious functionality as legitimate software, often opening backdoors for attackers. Only install software from trusted sources and validate signatures when available.

Ransomware & Spyware

Ransomware encrypts files for extortion; spyware quietly steals information. Regular backups, least-privilege accounts, and endpoint protection help mitigate these threats.
